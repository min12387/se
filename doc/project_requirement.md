# 1. 서론

## 1.1 문서의 목적
본 문서는 Mini Drive 시스템 개발에 필요한 요구사항을 체계적으로 정의하기 위해 작성되었다.  
시스템이 제공해야 할 기능과 성능, 보안 및 인터페이스 요소를 명확히 규정하여 개발 과정의 기준으로 활용한다.

---

# 2. 시스템 개요

## 2.1 시스템 정의
Mini Drive는 조직 내 파일을 중앙 서버에서 관리하고, 사용자 간 파일 공유 및 협업을 지원하는 클라우드 기반 파일 관리 플랫폼이다.

---

## 2.2 주요 기능

| 기능 코드 | 기능 설명 |
|-----------|----------|
| MD-01 | 다양한 형식의 파일 업로드 및 다운로드 |
| MD-02 | 계층형 폴더 구조 생성 및 관리 |
| MD-03 | 사용자 및 그룹 기반 파일 공유 |
| MD-04 | 파일 검색 및 필터링 기능 |
| MD-05 | 파일 변경 이력 관리 및 복구 |

---

## 2.3 용어 설명

| 용어 | 의미 |
|------|------|
| 접근 권한 | 사용자에게 허용된 파일 접근 범위 |
| 공유 링크 | 외부 사용자 접근을 위한 URL |
| 버전 관리 | 파일 변경 내역을 저장하고 이전 상태로 복구하는 기능 |

# 3. 요구사항 정의

## 3.1 기능 요구사항

### 3.1.1 사용자 계정 관리

| ID | 요구사항 | 중요도 |
|----|----------|--------|
| FR-01 | 사용자는 이메일 기반으로 회원가입 및 로그인을 수행할 수 있다 | 높음 |
| FR-02 | 관리자는 사용자 계정을 생성 및 삭제할 수 있다 | 높음 |
| FR-03 | 관리자는 사용자별 저장 용량을 설정할 수 있다 | 중간 |

---

### 3.1.2 파일 및 폴더 처리

| ID | 요구사항 | 중요도 |
|----|----------|--------|
| FR-04 | 사용자는 파일을 업로드 및 다운로드할 수 있다 | 높음 |
| FR-05 | 사용자는 폴더를 생성하고 구조를 변경할 수 있다 | 높음 |
| FR-06 | 시스템은 삭제된 파일을 일정 기간 보관한다 | 중간 |

---

### 3.1.3 공유 및 협업 기능

| ID | 요구사항 | 중요도 |
|----|----------|--------|
| FR-07 | 사용자는 파일 공유 시 접근 권한을 설정할 수 있다 | 높음 |
| FR-08 | 시스템은 외부 접근용 공유 링크를 생성한다 | 높음 |
| FR-09 | 공유 링크에 만료 시간을 설정할 수 있다 | 중간 |

---

### 3.1.4 검색 및 관리 기능

| ID | 요구사항 | 중요도 |
|----|----------|--------|
| FR-10 | 사용자는 파일 이름으로 검색할 수 있다 | 높음 |
| FR-11 | 검색 시 날짜 및 유형 필터를 적용할 수 있다 | 중간 |
| FR-12 | 시스템은 파일 변경 이력을 기록한다 | 중간 |

---

## 3.2 비기능 요구사항

### 3.2.1 시스템 환경

| ID | 요구사항 | 중요도 |
|----|----------|--------|
| NFR-01 | 시스템은 웹 기반으로 동작해야 한다 | 높음 |
| NFR-02 | 별도 프로그램 설치 없이 접근 가능해야 한다 | 높음 |

---

### 3.2.2 성능 요구사항

| ID | 요구사항 | 중요도 |
|----|----------|--------|
| NFR-03 | 200명 이상의 동시 접속을 지원해야 한다 | 높음 |
| NFR-04 | 파일 검색 및 다운로드는 빠른 응답을 제공해야 한다 | 높음 |

---

### 3.2.3 보안 요구사항

| ID | 요구사항 | 중요도 |
|----|----------|--------|
| NFR-05 | 데이터 전송 시 암호화 통신을 적용해야 한다 | 높음 |
| NFR-06 | 사용자 권한에 따라 접근이 제한되어야 한다 | 높음 |

---

## 3.3 인터페이스 요구사항

- IF-01: 웹 브라우저 기반 UI 제공 (Chrome, Edge 등 지원)  
- IF-02: 파일 업로드 시 드래그 앤 드롭 기능 지원  
- IF-03: 사용자 친화적인 화면 구성 제공
