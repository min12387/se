# 프로젝트 정의서

## 1. 프로젝트 명칭
기존보다 더 혁신적인 파일 관리 시스템

---

## 2. 개발 시스템 명칭
Mini Drive

---

## 3. 개발 시스템 설명
- 파일의 업로드와 다운로드를 가능하게
- 팀원 간의 파일 공유를 가능하게
- 파일 검색을 가능하게
- 파일 보안을 더 철저하게
---

## 4. 핵심 기능

| 기능 |
|------|
| 파일 업로드 및 다운로드 |
| 파일 공유 및 관리 |
| 보안성 및 접근 권한 관리 |
| 파일 검색과 탐색 기능|
| 파일의 활동 로그 |
