# 프로젝트 정의서

## 1. 프로젝트 명칭
Mini Drive — 클라우드 파일 공유 시스템

---

## 2. 개발 시스템 명칭
Mini Drive Cloud File Sharing System

---

## 3. 개발 시스템 설명
Mini Drive는 사용자 간 파일을 업로드하고 공유할 수 있는 클라우드 기반 파일 공유 시스템이다.  
사용자는 파일을 저장하고 다운로드할 수 있으며, 권한 설정을 통해 파일 접근을 제어할 수 있다.  
또한 협업을 위한 기능을 제공하여 여러 사용자가 효율적으로 작업을 수행할 수 있도록 한다.

---

## 4. 핵심 기능

| 기능 | 설명 |
|------|------|
| 파일 업로드 및 다운로드 | 다양한 파일을 업로드하고 다운로드 가능 |
| 파일 공유 | 사용자 간 파일 공유 기능 제공 |
| 접근 권한 관리 | 사용자별 읽기/쓰기 권한 설정 |
| 파일 검색 | 파일 이름 및 정보로 검색 |
| 활동 로그 | 파일 접근 및 수정 기록 확인 |

---

## 5. 기존 유사 소프트웨어 조사 및 분석

### 1) Google Drive
- 장점: 실시간 협업, 다양한 기능 제공
- 단점: 일부 기능은 유료

### 2) Dropbox
- 장점: 간단한 UI, 빠른 동기화
- 단점: 기능이 비교적 제한적

### 3) OneDrive
- 장점: Microsoft 서비스와 연동
- 단점: 환경에 따라 성능 차이 존재

---

## 6. 차별성
- 간단한 UI로 사용자 접근성 향상
- 필수 기능 중심으로 경량화
- 직관적인 파일 관리 구조 제공
