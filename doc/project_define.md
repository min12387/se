# 프로젝트 정의서

## 1. 프로젝트 명칭
기존보다 더 혁신적인 파일 관리 시스템

---

## 2. 개발 시스템 명칭
Mini Drive

---

## 3. 개발 시스템 설명
- 파일의 업로드와 다운로드 기능
- 팀원 간의 파일 공유 기능
- 파일 검색 기능
- 파일 보안 기능
---

## 4. 핵심 기능

| 기능 |
|------|
| 파일 업로드 및 다운로드 |
| 파일 공유 및 관리 |
| 보안성 및 접근 권한 관리 |
| 파일 검색과 탐색 기능|
| 파일의 활동 로그 | 

---

## 5. 기존 유사 소프트웨어 조사 및 분석

### 1) Google Drive
- 장점: 실시간 협업, 다양한 기능 제공
- 단점: 일부 기능은 유료

### 2) Dropbox
- 장점: 간단한 UI, 빠른 동기화
- 단점: 기능이 비교적 제한적

### 3) OneDrive
- 장점: Microsoft 서비스와 연동
- 단점: 환경에 따라 성능 차이 존재

---

## 6. 차별성
- 간단한 UI로 사용자 접근성 향상
- 필수 기능 중심으로 경량화
- 직관적인 파일 관리 구조 제공
